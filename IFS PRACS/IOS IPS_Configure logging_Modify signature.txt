Q.20 Create the following topology to Configure IOS Intrusion Prevention System (IPS)
Using the CLI.


==========Enable the Security License & Create the Directory===========================

R1(config):
license boot module c1900 technology-package securityk9
exit
show version
write memory
reload [y]

R1#
mkdir ipsdir   [enter whe prompted]


R1(config):
ip ips config location flash:ipsdir
ip ips name iosips
ip ips notify log
service timestamps log datetime msec
logging host 192.168.1.50

R1(config):
ip ips signature-category
category all
retired true
exit
category ios_ips basic
retired false
exit
exit  
[Press Enter when prompted]

R1(config):
interface gig0/0
ip ips iosips out
exit

R1(config):
ip ips signature-definition
signature 2004 0
status
retired false
enabled true
exit
engine
event-action produce-alert
event-action deny-packet-inline
exit
exit
exit
[Press Enter when prompted]
press exit until you reach R1#

================================Verification Testing=================================

ping 192.168.1.2   [fail]

PCA
ping 192.168.3.2  [pass]

Syslog Server -> Services tab -> SYSLOG   
You should see the IPS alert logs
